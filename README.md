# 🎲 DungeonTrack

**DungeonTrack** is a web application designed to help tabletop RPG players and game masters organize their campaigns, track session history, and manage characters and players in a structured way.

The goal of the project is to create a centralized place where RPG groups can document their adventures, maintain campaign continuity, and keep important information easily accessible.

## ⚔️ Features (Planned)

DungeonTrack aims to provide tools for organizing RPG campaigns, including:

- 📜 **Session History** – Record and review past game sessions
- 🧝 **Character Management** – Store character details and progression
- 👥 **Player Management** – Track players participating in campaigns
- 🏰 **Campaign Organization** – Keep campaigns structured and easy to navigate
- 🗂 **Adventure Logs** – Maintain a chronological record of events
- 🔍 **Search and Filters** – Easily find characters, sessions, or campaigns

More features will be added as the project evolves.

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/brunomachadors/dungeontrack.git
```

Navigate to the project folder:

```bash
cd dungeontrack
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

The application will start at:

```
http://localhost:3000
```
